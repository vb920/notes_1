# My Final Recommendation

For your actual roadmap:

## Phase 1 — First 100 topics

Do topics **1–100** deeply.  
This is enough for a huge number of Codeforces Div2/Div1-A/B/C style problems.

## Phase 2 — Topics 101–150

Add DP optimizations and number theory.  
This helps with harder Div2 D/E and Div1 lower problems.

## Phase 3 — Topics 151–200

Add advanced number theory, combinatorics, and strings.

## Skip for now

Anything that appears in Um\_nik’s “things I don’t know” list should be treated as:

> “Learn only when I see 3–5 problems where this exact tool is necessary.”

That is the right mindset.
